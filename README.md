# 2D-game-Universe-Development
This is a Software Game Development project.
I am using Unity Engine for this 2D Game called "Astro Dodge".
It is my first time with Unity Engine and C# (C-Sharp) coding, hence I am being guided by Unity Learn Tutorial.
The game uses a Univeral 2D environment.
# Basic Game Idea
You are a Rocket/Spaceship in space and you have to dodge various types of continuously evolving obstacles!
More you score, the Astroids aka obstacles get harder and faster.
Enjoy!
